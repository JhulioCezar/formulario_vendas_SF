# Sistema de Abertura de Chamados

Sistema web para abertura de chamados de novos clientes com opções de débito em conta e boleto bancário.

## Funcionalidades

- ✅ Formulário responsivo para desktop e mobile
- ✅ Duas opções de pagamento: Débito em Conta e Boleto
- ✅ Validação completa de campos
- ✅ Formatação automática (CPF, CEP, telefone, moeda)
- ✅ Geolocalização para preenchimento automático de endereço
- ✅ Upload de imagem do documento com câmera
- ✅ Copiar dados para área de transferência
- ✅ Salvar formulário como PDF
- ✅ PWA (Progressive Web App) com instalação
- ✅ Service Worker para funcionamento offline
- ✅ Atualizações automáticas

## Tecnologias Utilizadas

- HTML5
- CSS3 (com design moderno e responsivo)
- JavaScript (ES6+)
- PWA (Service Worker, Web App Manifest)
- jsPDF para geração de PDF
- HTML2Canvas para captura de tela
